
# 📘 Assignment: Hangman Game

## 🎯 Objective

Practice Python control flow, string handling, user input, and game logic by completing a Hangman game.

## 📝 Tasks

### 🛠️ Complete the Hangman game logic

#### Description
Finish the starter code to create a playable Hangman game with a hidden word, guessed letters display, and input handling.

#### Requirements
Completed program should:

- Randomly select a word from the provided list.
- Show current progress with underscores for unknown letters and revealed letters for correct guesses.
- Accept single-letter guesses and update the game state.
- Track incorrect guesses and remaining attempts.
- Prevent repeated guesses from changing the remaining attempts.


### 🛠️ Add end-game conditions and feedback

#### Description
Implement the final game outcome and display a clear win or lose message when the game ends.

#### Requirements
Completed program should:

- End when the player guesses the entire word or uses all allowed incorrect guesses.
- Print a winning message when the secret word is fully revealed.
- Print a losing message and reveal the secret word when attempts are exhausted.
- Display the number of guesses remaining after each incorrect attempt.
